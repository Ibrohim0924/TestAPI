# Проект для работы с тестовым API

Этот проект демонстрирует навыки работы с HTTP-коммуникацией и обработкой данных.

## О проекте

Программа выполняет следующие задачи:

1. Отправляет POST запрос для получения первой части кода
2. Получает вторую часть кода через webhook
3. Объединяет обе части кода
4. Отправляет GET запрос для получения итогового результата

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Ibrohim0924/TestAPI.git
cd TestAPI
```

2. Установите зависимости:
```bash
npm install
```

3. Получите URL с webhook.site:
   - Перейдите на сайт [webhook.site](https://webhook.site/)
   - Скопируйте полученный уникальный URL
   - Вставьте URL в переменную `requestData.url` в файле `index.js`

## Использование

1. Запустите программу:
```bash
npm start
```

2. Программа работает в следующем порядке:
   - Отправляется первый POST запрос и получается первая часть кода (`part1`)
   - На webhook.site появляется новый запрос, откуда нужно взять вторую часть кода (`part2`)
   - Введите `part2` в программу
   - Программа объединяет обе части и получает финальный результат через GET запрос

## Техническая информация

- **API URL**: https://test.icorp.uz/interview.php
- **Типы запросов**:
  - POST: Для получения первой части кода
  - GET: Для получения финального результата
- **Webhook**: Для получения второй части кода

## Использованные технологии

- Node.js
- Axios (для HTTP запросов)
- Webhook.site (для callback URL)

## Устранение неполадок

Если при запуске программы возникают ошибки:

1. Проверьте подключение к интернету
2. Убедитесь, что URL webhook.site введен правильно
3. Проверьте правильность копирования всех частей кода
